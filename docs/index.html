Spotlight Focus

Enhance your presentations with a cursor-following spotlight effect

Version 3.3 • Manifest V3

Overview

Spotlight Focus is a Chrome extension that dims the entire page and projects a soft spotlight wherever your cursor moves. Perfect for presentations, screen sharing, and live demos where you need to draw your audience's attention to specific parts of the screen.

Installation

Install in Developer Mode:

  1. Download or clone the repository from GitHub
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable Developer mode using the toggle in the top-right corner
  4. Click Load unpacked
  5. Select the source folder (containing manifest.json)
  6. The 🔦 Spotlight Focus icon will appear in your Chrome toolbar
Note: After installing, go to chrome://extensions/shortcuts to confirm that Alt+Shift+D is assigned to "Toggle spotlight on/off". If empty, click the field and press Alt+Shift+D to set it.

How to Use

  1. Click the 🔦 icon in your toolbar to open settings
  2. Click Enable Spotlight or press Alt+Shift+D
  3. Move your cursor to see the spotlight effect
  4. Adjust brightness and size sliders in the popup
  5. Hold Ctrl and scroll to resize while active
  6. Toggle off when you're done presenting
